# 🛍️ Sales Data Analysis for a Retail Store

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a retail sales dataset to uncover useful business insights.
The analysis helps answer questions such as:

* Which **month generates the highest sales**
* Which **cities contribute most to revenue**
* What is the **best time to display advertisements**
* Which **products are frequently purchased together**

The project demonstrates **data cleaning, transformation, visualization, and business insight generation using Python**.

## 🚀 How to Run the Project

### 1️⃣ Clone the repository

```bash
git clone https://github.com/tanu91112/sales-data-analysis.git
cd sales-data-analysis
```

### 2️⃣ Install dependencies

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### 3️⃣ Run the notebook

```bash
jupyter notebook
```

Open the **Sales_Data_Analysis.ipynb** file and run all cells to see the analysis and visualizations.
